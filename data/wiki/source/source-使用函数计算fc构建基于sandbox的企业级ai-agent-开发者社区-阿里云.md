# 来源：使用函数计算FC构建基于Sandbox的企业级AI Agent-开发者社区-阿里云

## 文档概要
本文发布于阿里云开发者社区（2025-09-16），面向企业开发者，提供一种基于 Serverless 与 Sandbox 深度融合的 AI Agent 构建路径。

## 核心主张
- AI Agent 应摆脱‘黑盒’实验形态，成为企业可精准掌控的生产力工具；
- Sandbox 与 Serverless 的结合构成安全、可控、可扩展的技术底座；
- 该方案适配 Chat Coding 类动态交互场景，并为多模态 Agent、跨系统协作等未来需求预留扩展性。

## 关键技术依赖
- 底层运行环境：[[aliyun-fc]] 提供的函数计算能力；
- 安全机制：[[sandbox-isolation]] 实现的沙箱隔离；
- 架构范式：[[ai-agent-architecture]] 所定义的动态交互架构；
- 目标实体：所构建的 [[ai-agent]] 具备企业级落地能力。
