# 可观测性

## 三大支柱
- **日志（Logs）**：记录 Agent 每次执行的输入、输出、中间步骤与异常；
- **指标（Metrics）**：统计调用次数、延迟、错误率、Token 消耗等量化数据；
- **链路追踪（Tracing）**：还原 Agent 决策路径，包括 LLM 调用、工具调用、子 Agent 协作等环节。

## 在 AgentRun 中的体现
- 全链路自动注入 Trace ID，跨函数、跨服务串联；
- 默认集成阿里云 SLS 与 ARMS，支持自定义仪表盘与告警；
- 为 [[生产级部署]] 提供根因分析与性能优化依据。
